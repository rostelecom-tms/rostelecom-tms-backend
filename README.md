# Rostelecom TMS Backend

Система управления тестами

---

Что уже реализовано:

- Сущности тест-кейсов, групп, проектов, пользователей, прогонов, дефектов, тэгов + репозитории, сервисы и контроллеры для них
- Дэшборд
- Вебхук для загрузки результатов прогонов
- Запуск фронта и бэка в docker-контейнерах
- Кэширование (Redis)
